# kaggle-taste-labs

Experiments with recommendation and preference learning on Kaggle datasets, exploring how machine learning can be used in innovative ways to understand taste and increase user engagement.

## Goals

- Build small, focused notebooks around music, media, and content-preference datasets.
- Use unsupervised methods (NMF, clustering, dimensionality reduction) to discover latent taste structures.
- Train simple, interpretable predictive models that connect early user signals to long-term preferences and engagement.
- Document each experiment with a short narrative: question, data, method, key findings.

## Notebooks (planned)

- Last.fm taste clusters and user engagement signals
- Digit and image-based preference toy problems
- Additional Kaggle recommendation datasets (to be added)

## How to use this repo

Each notebook is self-contained and designed as a small lab. You can run them on Kaggle or locally; the README of each notebook folder will point to the original dataset and outline the main learning ideas.
